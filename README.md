# HomeTown Design & Build

## Installation

Install Gatsby Globally
```
npm install -g gatsby
```

## Installing Dependencies

Go to working directory and run

```
npm install
```

## Development

```
gatsby develop
```

## Build

For Building the project just run

```
gatsby build
```

## Server

For Serving Public folder just run

```
gatsby serve
```
